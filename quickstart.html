🚀 快速开始 ​

本文档将指导您如何快速开始使用 NotebookLM2PPT 工具，将 PDF 转换为高质量的可编辑 PowerPoint 演示文稿。

📋 系统要求 ​

  • 操作系统: Windows 10/11
  • Microsoft PowerPoint: 用于生成和合并幻灯片
  • 微软电脑管家: 版本 ≥ 3.17.50.0

📦 安装方法 ​

推荐：下载预编译版本 ​

  1. 前往 Releases 页面
  2. 下载最新的 .exe 执行文件
  3. 双击运行即可，无需安装 Python 环境

其他安装方式 ​

通过 pip 安装：

bash
pip install notebooklm2ppt -U

从源码安装：

bash
pip install git+https://github.com/elliottzheng/NotebookLM2PPT.git

⚙️ 配置微软电脑管家 ​

  1. 打开微软电脑管家 -> 工具箱
  2. 找到 智能圈选，确保其已开启
  3. 确认快捷键为 Ctrl + Shift + A
  4. 测试：按下快捷键，确保能看到圈选界面并出现“转换为PPT”选项

📝 基础使用流程 ​

步骤 1：启动程序 ​

  • 双击下载的 .exe 文件，或
  • 在命令行中运行：notebooklm2ppt

步骤 2：选择文件 ​

  • 点击"浏览"按钮选择要转换的 PDF 文件
  • （可选）选择对应的 MinerU JSON 文件以获得更好的转换效果

步骤 3：设置参数 ​

对于首次使用，建议使用默认参数：

  • DPI: 200（控制图片质量）
  • 显示比例: 0.8（控制图片大小）
  • 延迟: 2 秒（控制操作间隔）
  • 超时: 60 秒（控制等待时间）

步骤 4：校准按钮位置（首次使用必须） ​

  1. 勾选"校准按钮位置"选项
  2. 程序会显示智能圈选界面
  3. 手动点击"转换为PPT"按钮
  4. 程序会自动保存偏移值，下次使用时无需重复校准

步骤 5：开始转换 ​

  1. 点击"开始转换"按钮
  2. 程序会自动执行转换流程
  3. 转换过程中请勿操作鼠标键盘
  4. 转换完成后，会在 workspace 文件夹生成 PPT 文件

🧠 使用 MinerU 优化 ​

步骤 1：获取 MinerU JSON ​

  1. 访问 MinerU 官网
  2. 上传您的 PDF 文件
  3. 等待解析完成（通常需要几分钟）
  4. 下载生成的 JSON 文件

步骤 2：使用 JSON 文件 ​

  1. 在程序中选择 PDF 文件
  2. 选择对应的 JSON 文件
  3. 点击"开始转换"按钮
  4. 程序会在基础转换后执行 MinerU 优化

🖼️ 仅图片模式 ​

对于快速转换，可使用仅图片模式：

  1. 勾选"仅图片模式"选项
  2. 选择要转换的 PDF 文件
  3. 点击"开始转换"按钮
  4. 程序会直接将图片插入 PPT，速度更快但内容不可编辑

🎨 图像修复方法 ​

在程序中，您可以选择不同的图像修复方法：

  • 智能平滑（推荐）：综合效果最佳，适合大多数场景
  • 边缘均值填充：适合纯色背景
  • 极速纯色填充：适合极简背景，速度最快
  • 逐层内缩修补：适合细长划痕或线条修复
  • 渐变过渡插值：适合带有渐变的背景
  • 双调和光影修补：保持光影连续性，速度较慢

⚠️ 注意事项 ​

  • 保持窗口焦点：转换过程中请勿操作鼠标键盘
  • 检查下载目录：确保临时文件能正确保存到系统下载文件夹
  • 更新软件：确保微软电脑管家已更新到最新版本
  • 网络连接：使用 MinerU 优化时需要稳定的网络连接

📊 常见问题 ​

Q: MinerU JSON 文件是必需的吗？ A: 不是必需的。不提供 JSON 文件时，程序会使用基础转换功能。

Q: MinerU 是免费服务吗？ A: 是的，目前 MinerU 是免费服务，只需要注册即可使用。

Q: 转换速度太慢怎么办？ A: 可以尝试降低 DPI 参数、使用仅图片模式或关闭其他占用资源的应用。

Q: 生成的 PPT 中图片质量不好怎么办？ A: 可以提高 DPI 参数（如设置为 300）或使用 MinerU 优化获取原始高清图片。

🚀 最佳实践 ​

  1. 首次使用：

    • 确保微软电脑管家已更新到最新版本
    • 校准按钮位置
    • 使用默认参数进行测试

  2. 参数优化：

    • 对于一般文档，使用默认参数即可
    • 对于高质量要求，将 DPI 设置为 300
    • 对于大型文档，适当增加延迟和超时时间

  3. 流程选择：

    • 对于重要文档，使用 MinerU 优化
    • 对于快速转换，使用仅图片模式
    • 对于复杂背景，选择智能平滑修复方法

  4. 结果检查：

    • 转换完成后检查生成的 PPT
    • 确认文本框、图片和背景是否正确
    • 根据需要进行手动调整

通过遵循以上指南，您可以快速上手 NotebookLM2PPT，将 PDF 转换为高质量的可编辑 PowerPoint 演示文稿。
