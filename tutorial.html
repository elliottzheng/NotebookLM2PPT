使用教程 ​

本教程将详细介绍如何使用 NotebookLM2PPT 工具，从基础使用到高级功能。

📖 基础使用教程 ​

步骤 1：准备工作 ​

  1. 安装微软电脑管家

    • 前往 微软电脑管家官网 下载并安装
    • 确保版本 ≥ 3.17.50.0

  2. 配置智能圈选

    • 打开微软电脑管家 → 工具箱 → 智能圈选
    • 确保快捷键为 Ctrl + Shift + A
    • 测试智能圈选功能，确保能看到"转换为PPT"选项

  3. 安装 NotebookLM2PPT

    • 下载预编译版本（推荐）或通过 pip 安装

步骤 2：转换 PDF ​

  1. 启动程序

    • 双击执行文件或运行 notebooklm2ppt 命令

  2. 选择文件

    • 点击"浏览"按钮选择要转换的 PDF 文件
    • （可选）选择 MinerU JSON 文件以获得更好的效果

  3. 设置参数

    • DPI：控制图片质量，默认 200
    • 显示比例：控制图片大小，默认 0.8
    • 延迟：控制操作间隔，默认 2 秒
    • 超时：控制等待时间，默认 60 秒

  4. 开始转换

    • 点击"开始转换"按钮
    • 程序会自动执行转换流程
    • 转换完成后，会在 workspace 目录生成 PPT 文件

🚀 高级功能教程 ​

MinerU 优化使用 ​

  1. 获取 MinerU JSON

    • 访问 MinerU 官网
    • 上传 PDF 文件
    • 等待解析完成
    • 下载生成的 JSON 文件

  2. 使用 JSON 文件

    • 在程序中选择 PDF 文件
    • 选择对应的 JSON 文件
    • 点击"开始转换"
    • 程序会在基础转换后执行 MinerU 优化

仅图片模式使用 ​

  1. 启动程序
  2. 勾选"仅图片模式"选项
  3. 选择 PDF 文件
  4. 点击"开始转换"
  5. 程序会跳过智能圈选，直接将图片插入 PPT

图像修复方法选择 ​

  1. 启动程序
  2. 在"图像修复方法"下拉菜单中选择合适的方法
  3. 点击"开始转换"

推荐方法：

  • 智能平滑：综合效果最佳，适合大多数场景
  • 边缘均值填充：适合纯色背景
  • 极速纯色填充：适合极简背景，速度最快

💡 使用技巧 ​

1. 提高转换质量 ​

  • 增加 DPI：对于高质量要求，将 DPI 设置为 300
  • 使用 MinerU 优化：获取原始高清图片
  • 选择合适的修复方法：根据背景复杂度选择

2. 提高转换速度 ​

  • 降低 DPI：对于快速转换，将 DPI 设置为 150
  • 使用仅图片模式：跳过智能圈选，速度更快
  • 减小显示比例：减少屏幕显示时间

3. 解决常见问题 ​

  • 按钮位置校准：首次使用时必须校准
  • 保持窗口焦点：转换过程中不要操作鼠标键盘
  • 检查下载目录：确保临时文件能正确保存

4. 批量处理 ​

  1. 准备多个 PDF 文件
  2. 逐个转换：目前不支持批量转换，需要逐个处理
  3. 统一设置：使用相同的参数设置确保一致性

🎯 最佳实践 ​

场景 1：NotebookLM 演示文稿 ​

  1. 生成 PDF：在 NotebookLM 中生成演示文稿 PDF
  2. 获取 JSON：上传到 MinerU 获得 JSON 文件
  3. 转换优化：使用默认参数 + MinerU 优化
  4. 检查结果：确保文本框和图片正确

场景 2：快速转换 ​

  1. 选择文件：选择要转换的 PDF
  2. 使用仅图片模式：跳过智能圈选
  3. 快速完成：适合对编辑性要求不高的场景

场景 3：高质量转换 ​

  1. 设置 DPI 为 300：提高图片质量
  2. 获取 MinerU JSON：获得最佳优化效果
  3. 选择智能平滑修复：保持背景质量
  4. 耐心等待：高质量转换需要更多时间

📚 常见问题解答 ​

Q: 转换过程中程序无响应怎么办？ A: 这是正常现象，程序正在执行自动化操作。请勿强制关闭，等待转换完成。

Q: 生成的 PPT 中图片质量不好怎么办？ A: 提高 DPI 参数，或使用 MinerU 优化获取原始高清图片。

Q: 智能圈选没有"转换为PPT"选项怎么办？ A: 确保微软电脑管家已更新到最新版本，检查是否开启了预览体验计划。

Q: MinerU JSON 文件是必需的吗？ A: 不是必需的。但使用 JSON 文件可以获得更好的优化效果。

Q: 转换速度太慢怎么办？ A: 尝试降低 DPI、使用仅图片模式、关闭其他应用程序。

通过本教程，您应该能够熟练使用 NotebookLM2PPT 工具，将 PDF 转换为高质量的可编辑 PowerPoint 演示文稿。
